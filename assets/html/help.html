Searching

To search, first select which of the three datasets you are searching using the drop-down, then enter your search terms into the text field.

Fields are searched for with fieldname:query. By default, all fields are searched, including data that isn't indexed by name, for example, publication page numbers from the references dataset.

Complete phrases can be matched by using double quotes (") around the phrase. For example, you could search for "lower cambrian" to find species occurances in the lower cambrian.

The asterisk (*) can be used to perform wildcard matches. To match any genus that begins with "poly" you can search for genus:poly* Since the wildcard matches any data, it can be used to find just species with images, via images:*

The plus symbol (+) means required, so if you want to search for species published by Robert Ravn AND those with images, you'd search for +author:ravn +images:*

The minus symbol (-) means do not match. To find all species published by Robert Ravn but without images, enter author:ravn -images:*

Range queries allow for numeric range searches. If you want a reference published between 1997 and 1999, search for +year:>=1997 +year:<=1999

Example Queries

cambrian matches any reference that contains the word "cambrian"
"lower cambrian" matches any reference that contains the phrase "lower cambrian"
"lower cambrian" -acritarchs matches any reference that contains the phrase "lower cambrian" but does not contain "acritarchs"
genus:poly* matches any genus that begins with "poly"
images:* matches only species with images
author:brown matches any author with the name of "brown"
author:ravn images:* matches species authors named "ravn" OR any species with images
+author:ravn +images:* matches species authors named "ravn" AND any species with images
+year:>=1997 +year:<=2000 matches any genus published between 1997 and 2000
