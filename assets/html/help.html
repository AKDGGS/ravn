Searching

Fields are searched for with "fieldname:query". By default, all fields are searched, including data that isn't indexed by name, for example, publication page numbers from the references database.

The asterisk (*) can be used to perform wildcard matches. To match any genus that begins with "poly" you can search for "genus:poly*" Since the wildcard matches any data, it can be used to find just species with images, via "images:*"

The plus symbol means required, so if you want to search for species done by Robert Ravn with images, you'd search for "+author:ravn +images:*"

By contrast, "author:ravn images:*" will search for species that have images or were authored by Robert Ravn

Range queries allow for numeric range searches. If you want a reference published between 1997 and 1999, search for "+year:>=1997 +year:<=1999"
